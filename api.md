# API documentation

## Introduction 

Api are the bone of all our plugins, they contain reusable code (in theory) who are at the core of all custom mecanics we developed. It essential to know all already existing API that exist so we don't redo the same code over and over again. You'll find right there all API and what they do, use them as you wish.

## API

### [InventoryALaCarte](api/inventoryALaCarte/main.md)

InventoryALaCarte is an API that handle custom menu. With it you can create custom data driven menu, display them to any player and handle what they do !
